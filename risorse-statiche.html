Risorse statiche

Le risorse statiche come immagini, fogli d stili o pagine web statiche, devono essere cofigurate per essere usate in modo corretto all'interno delle pargine dinamiche generato dall'express.

Configurazione

const express = require('express');
const app = express();
const port = 3000;

//per mappare su /
app.use(express.static("/static"));

//per mappare su un alias
app.use("/alias", express.static("/static"));

Possiamo configurare le nostre risorse statiche ad essere mappato al root ("/"), o ad un alias
