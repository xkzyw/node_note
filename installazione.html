Linux

Le distribuzioni Linux hanno differente package manager system, quindi si deve cercare il commando giusto in base alla proria distribuzione.

Arch Linux based

  1. Aggiornare il gestore del pacchetto

    sudo pacman -Syy

  2. Installare nodejs e npm

    sudo pacman -S nodejs npm

  3. Verificare l'installazione di nodejs e npm

    node -v

Debian Linux based

  1. Aggiornare il gestore del pacchetto

    sudo apt update

  2. Installare nodejs e npm

    sudo apt install nodejs npm

  3. Verificare l'installazione di nodejs e npm

    node -v
...
npm -v
...

Windows

  1. Scaricare Node.js

    Scarica l'installer dal sito ufficiale dei nodejs.

  2. Eseguire l'installer

    Dopo il download, esegui l'installer e segui le istruzioni guidate. Di solito, le impostazioni predefinite sono appropriate per la maggior parte degli utenti.

  3. Verificare l'installazione

    Dopo l'installazione, apri il Prompt dei Comandi o PowerShell e esegui i seguenti comandi per verificare che Node.js e npm (Node Package Manager) siano stati installati correttamente: 

    node -v
...
npm -v
...
