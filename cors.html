CORS

Il termine "CORS" sta per "Cross-Origin Resource Sharing" ed è una politica di sicurezza implementata nei browser web per controllare come le risorse di una pagina web possono essere richieste da un dominio diverso da quello della pagina stessa.

Quando una pagina web fa una richiesta HTTP da un dominio A a un dominio B (ad esempio, quando una pagina web su http://dominioA.com fa una richiesta a http://dominioB.com tramite JavaScript), la politica di sicurezza del browser può bloccare la richiesta, a meno che il server al dominio B non fornisca le autorizzazioni appropriate.

Il problema principale che CORS cerca di risolvere è la potenziale minaccia di sicurezza quando si consente a pagine web di eseguire richieste da un dominio diverso. Questa politica impedisce richieste non autorizzate e protegge gli utenti da potenziali attacchi.

Gestire CORS

Nell'express, per gestire CORS si usa il modulo cors().

Dato che questo modulo non è fornito dall'express , dobbiamo installarlo attraverso npm install. 

npm install cors
const express = require('express');
const app = express();
const port = 3000;

// Usare il modulo cors()
app.use(cors());
app.options("*", cors());

app.listen(port, () => {
    console.log(`Server on http://localhost:${port}`);
});
