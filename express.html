express

Express.js è un framework web per Node.js che semplifica lo sviluppo di applicazioni web fornendo un set di funzionalità robuste per la gestione delle route, delle richieste HTTP e delle risposte.

Express.js è ampiamente utilizzato per la creazione di applicazioni web e API server-side in Node.js a causa della sua semplicità e flessibilità.

Installazione

Si deve utilizzare il commando npm install per installare express 

npm install express

Inizializzazione

Si deve prima inizializzare express prima di utilizzarlo. 

const express = require('express');
const app = express();
const port = 3000;

app.listen(port, () => {
    console.log(`Server on http://localhost:${port}`);
});

Durante l'inizializzazione viene creato una istanza del classe Express, il quale contiene il metodo listen() che ha bisogno di due parametri, la porta dove ascoltare le richieste del client e un callback richiamato nel momento che il server è in ascolto.

Avviamento

Infine, per avviare il server si deve utilizzare il commando node seguito dal percorse del file di Javascript. 

node [file].js
